## Stack


- Laravel 13 + Filament 5
- PHP 8.4
- MySQL 8
- Docker + Docker Compose
- Livewire

## Correr el proyecto localmente

```bash
git clone <url>
cd ticketwave
git checkout develop
cp .env.example .env
make setup
```

La app estará en `http://localhost:8000`
